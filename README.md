
# คู่มือการใช้งาน Calculator-may

สร้างไว้ใช้บวกเลขชิลล์ๆ

สร้างตอนเรียน Node.js

- สามารถบวกเลขได้
- ลบได้
- คูณได้

## วิธีใช้งาน

ต้องลง NodeJS ก่อน [ดาวน์โหลดได้ที่นี่](https://nodejs.org/en/)

วิธี require module

```js
const calculator = require('calculator-may')
```
### วิธีการบวก
สำหรับการบวก    function `plus()`

```js
calculate.plus(2,3);
```
